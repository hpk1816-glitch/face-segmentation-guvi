# Face Segmentation - Movie Cast Identification 
 
## Project Overview 
Real-Time Face Segmentation using U-Net + MobileNetV2 for movie cast identification. 
 
## Tech Stack 
- Python, TensorFlow, OpenCV, Streamlit 
 
## Model 
- Architecture: U-Net with MobileNetV2 encoder 
- Loss: Dice Loss 
- Metrics: Dice Coefficient, IoU 
 
## How to Run 
pip install -r requirements.txt 
streamlit run app/app.py
